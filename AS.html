Rhinoceros

2D Rhinoceros
3D Rhinoceros

Proceso

- Colocamos la imagen con el comando _PictureFrame, luego se selecciona la imagen y el tamaño es a elección.

- Damos clic en la opción Layers (capas) / New Layer. Seleccionamos el color deseado, es recomendable un color que contraste las líneas de la imagen de fondo.

- Comenzamos a dibujar con la herrmienta _Curve solo se debe dibujar el contorno de la imagen.

- - Con el comando ExtrudeCrv seleccionamos las curvas que deseamos extruir.

Inventor

2D inventor

Proceso

- Create 2D sketch. Escogemos el plano en el que se va a dibujar.

- Luego hacemos uso de las herramientas. Para dejar de usar cada una se debe presionar la tecla esc.

- Si se quieres eliminar una entidad solo seleccionamos la curva y la suprimimos. En caso de solo una parte, Trim. - Usamos círculos para las curva y eliminamos las líneas no deseadas con trim.

SKECTHUP

3D SKECTHUP

Proceso

- Primero dibujamos un rectángulo con el tamaño de la cada luego con la herramienta Empujar/tirar, le damos altura al sólido. - Dibujamos rectángulos en la parte superior y lo empujamos hacia abajo. - Le damos color con la herramienta material.

3D sculptris

Proceso
